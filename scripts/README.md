# 📜 Операционные скрипты (Scripts)

## Описание
Директория содержит служебные скрипты для управления runtime-компонентами системы.

## Основные инструменты

*   **`llama-start.ps1`**: Управление запуском сервера `llama-server.exe` с поддержкой GGUF моделей.
    *   Автоматический поиск моделей на дисках.
    *   Настройка параметров инференса (потоки, GPU слои).
    *   Интеграция с API FastAPI Foundry для удаленного управления.

## Правила
Любые изменения в параметрах запуска моделей должны отражаться в конфигурации `.env`.
