# FastAPI Foundry - Product Overview

## Project Purpose
FastAPI Foundry is a modern REST API server designed to provide seamless access to local AI models through Foundry, with integrated Retrieval-Augmented Generation (RAG) capabilities. It serves as a bridge between local AI infrastructure and applications requiring intelligent text generation and context-aware responses.

## Value Proposition
- **Local AI Control**: Run AI models locally without cloud dependencies
- **RAG Integration**: Enhanced responses through document context retrieval
- **Developer-Friendly**: REST API with comprehensive documentation and web interface
- **Production-Ready**: Docker support, monitoring, and security features
- **Extensible**: MCP server integration for Claude Desktop and other clients

## Key Features

### Core AI Capabilities
- **Multi-Model Support**: DeepSeek, Qwen, Mistral, Llama models
- **Text Generation**: High-quality AI-powered text completion
- **RAG System**: Context-aware responses using document knowledge base
- **Batch Processing**: Handle multiple requests efficiently
- **Streaming Responses**: Real-time text generation

### API & Integration
- **REST API**: Full-featured FastAPI with automatic documentation
- **Web Interface**: Built-in chat and model management UI
- **MCP Server**: Integration with Claude Desktop and MCP clients
- **CORS Support**: Cross-origin requests for web applications
- **API Authentication**: Secure access through API keys

### Operations & Deployment
- **Docker Ready**: Complete containerization support
- **Health Monitoring**: Service and model status endpoints
- **Logging System**: Comprehensive request and error tracking
- **Multiple Launchers**: Various deployment scenarios supported
- **Tunnel Support**: Public access through ngrok/Cloudflare
