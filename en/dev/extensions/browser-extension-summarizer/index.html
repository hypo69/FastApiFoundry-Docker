    Browser Extension Summarizer

    Chrome-расширение (Manifest V3) для суммаризации веб-страниц, проверки фактов и чата с AI-моделями. Поддерживает 11 LLM-провайдеров и подключение к локальному серверу FastAPI Foundry.

    Провайдеры и модели Настройка API-ключей, загрузка моделей, Custom провайдер

    Быстрый старт

    1. chrome://extensions/ → Режим разработчика → Загрузить распакованное
2. Выбрать папку extensions/browser-extension-summarizer/
3. Popup → ⚙️ Providers & Models → добавить API-ключ
4. Правый клик на странице → Summarise → Only This Page
