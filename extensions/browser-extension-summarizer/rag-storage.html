📚 RAG Storage
ℹ️ Как работает:
1. ПКМ на любой странице → RAG Storage → Save Page to RAG — извлекает текст и сохраняет локально.
2. ПКМ → Open RAG Storage (или кнопка в popup) — открывает этот список.
3. Выбери записи чекбоксами, укажи URL сервера, нажми Send to RAG — отправит POST /api/v1/rag/build.
