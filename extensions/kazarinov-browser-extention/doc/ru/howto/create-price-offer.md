# Как собрать ценовое предложение

Это пошаговое руководство описывает полный процесс: от установки расширения до получения готового PDF с ценовым предложением.

---

## Шаг 1. Установка расширения

1. Откройте Chrome и перейдите на `chrome://extensions/`
2. Включите **Режим разработчика** (переключатель в правом верхнем углу)
3. Нажмите **Загрузить распакованное расширение**
4. Выберите папку с проектом расширения
5. Расширение появится в списке и на панели инструментов браузера

---

## Шаг 2. Настройка API ключа Gemini

1. Получите API ключ на [Google AI Studio](https://aistudio.google.com/app/apikey)
2. Нажмите на иконку расширения в панели инструментов Chrome
3. В открывшемся popup введите ваш **API ключ**
4. Выберите модель (по умолчанию `gemini-2.5-flash`)
5. Нажмите **Сохранить**

---

## Шаг 3. Сбор компонентов у поставщиков

Расширение поддерживает следующих поставщиков:
- `ksp.co.il`
- `ivory.co.il`
- `morlevi.co.il`
- `grandadvance.co.il`

Для каждого компонента:

1. Откройте страницу товара у поставщика в браузере
2. Кликните **правой кнопкой мыши** на странице
3. В контекстном меню выберите **Добавить компонент**
4. Расширение автоматически извлечёт данные (название, цену, характеристики) и сохранит компонент
5. Повторите для всех нужных комплектующих (процессор, материнская плата, ОЗУ, видеокарта и т.д.)

> Добавленные компоненты сохраняются в браузере и доступны между сессиями.

---

## Шаг 4. Просмотр сохранённых компонентов

После добавления хотя бы одного компонента в контекстном меню появится раздел **Сохраненные компоненты**. Там отображается список всех добавленных позиций. Каждый компонент можно удалить через подпункт **❌ Удалить**.

---

## Шаг 5. Генерация ценового предложения

1. Кликните правой кнопкой мыши на любой странице
2. Перейдите в **Сохраненные компоненты → Сформировать предложение цены**
3. Выберите язык итогового предложения (русский, английский, иврит и др.)
4. Расширение отправит данные в Gemini AI и откроет страницу предпросмотра

---

## Шаг 6. Предпросмотр и сохранение PDF

На странице предпросмотра отображается готовое ценовое предложение с:
- Названием и описанием сборки
- Списком компонентов с характеристиками
- Изображениями товаров

Для сохранения в PDF используйте стандартную печать браузера: `Ctrl+P` → выберите **Сохранить как PDF**.

---

## Управление компонентами

| Действие | Как выполнить |
|---|---|
| Добавить компонент | ПКМ на странице товара → **Добавить компонент** |
| Удалить один компонент | ПКМ → **Сохраненные компоненты** → компонент → **❌ Удалить** |
| Удалить все компоненты | ПКМ → **Сохраненные компоненты** → **💣 Очистить все компоненты** |
| Сформировать предложение | ПКМ → **Сохраненные компоненты** → **Сформировать предложение цены** → язык |
