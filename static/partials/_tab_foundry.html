Как скачать модель вручную через Foundry CLI
  1. Откройте PowerShell или Terminal.
  2. При необходимости посмотрите доступные модели: foundry model list
  3. Скачайте модель командой: foundry model download <model-id>
  4. Загрузите модель в сервис: foundry model load <model-id>
Пример: foundry model download qwen2.5-0.5b-instruct-generic-cpu:4, затем foundry model load qwen2.5-0.5b-instruct-generic-cpu:4
