Usage

llama.cpp exposes an OpenAI-compatible API.

After starting, add the Custom (OpenAI-compatible) provider in the browser extension:

http://127.0.0.1:8080/v1

PowerShell:

.\scripts\llama-start.ps1 -ModelPath "~/.models/gemma-2-2b-it-Q6_K.gguf"
Where to get GGUF models

GGUF is a binary model format for local inference, used by llama.cpp and compatible tools.

Where to download

1. Hugging Face (main source)

Search: gguf, llama gguf, mistral gguf. Files: model.Q4_K_M.gguf

  • TheBloke GGUF models
  • Mistral GGUF
  • LLaMA GGUF

2. LM Studio — built-in catalog, download via GUI.

3. Ollama — downloads automatically (ollama pull), but does not expose the file directly.

How to download

CLI (recommended):

pip install huggingface_hub
huggingface-cli download TheBloke/Mistral-7B-Instruct-GGUF \
  mistral-7b-instruct.Q4_K_M.gguf \
  --local-dir ./models

Direct:

wget https://huggingface.co/.../model.Q4_K_M.gguf

Via browser: open repository → select .gguf → download.

How to choose quantization
TypeRAMQuality
Q2minimallow
Q4_K_Mlowmedium
Q5_K_Mmediumgood
Q8highnear-original
  • up to 16 GB RAM → Q4_K_M
  • 16–32 GB → Q5
  • 32+ GB → Q8
Running
./main -m model.gguf -p "Hello"
