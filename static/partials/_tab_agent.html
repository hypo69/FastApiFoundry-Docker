AI Agent — модель вызывает инструменты

Эта вкладка демонстрирует возможности интеграции агентов, где модель может динамически вызывать инструменты для выполнения задач.

Пример:

# Python-код для примера
print("Hello from Agent!")
Notes

Для работы агентов необходимо наличие моделей, способных вызывать инструменты.

Например, модели, обученные на функциях OpenAI.
