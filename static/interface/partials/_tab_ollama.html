Install Ollama:

About Ollama

Ollama runs AI models locally with a simple CLI and REST API.

  • Fully local — no internet during inference
  • OpenAI-compatible /v1 endpoint
  • Supports Qwen, Llama, Mistral, DeepSeek, Gemma
  • Automatic GPU acceleration
