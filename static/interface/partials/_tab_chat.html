How models work in chat
  • List shows models from local caches.
  • On model switch, active models are unloaded.
  • To add: use Foundry, HuggingFace or llama.cpp tabs.
Load via CLI
  • Foundry: foundry model download deepseek-r1-distill-qwen-7b-generic-cpu:3 foundry model load deepseek-r1-distill-qwen-7b-generic-cpu:3
  • HuggingFace CLI: pip install huggingface_hub hf auth login hf download bartowski/gemma-2-2b-it-GGUF --local-dir ~/.models
  • llama.cpp: llama-server --model ~/.models/gemma-2-2b-it-Q6_K.gguf --port 9780 .\scripts\llama-start.ps1 -ModelPath ~/.models/gemma-2-2b-it-Q6_K.gguf
Get HuggingFace model via UI
  1. Open HuggingFace tab.
  2. For gated models set token in Settings.
  3. Enter model_id and click Download.
  4. Click Refresh in Downloaded Locally.
