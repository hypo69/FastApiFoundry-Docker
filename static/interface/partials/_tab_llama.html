Where to get GGUF models

GGUF is a binary model format for local inference, used by llama.cpp and compatible tools.

Where to download

1. Hugging Face (main source)

  • TheBloke GGUF models
  • Mistral GGUF
  • LLaMA GGUF

2. LM Studio — built-in catalog, download via GUI.

3. Ollama — ollama pull, but does not expose the file directly.

How to download
pip install huggingface_hub
huggingface-cli download TheBloke/Mistral-7B-Instruct-GGUF \
  mistral-7b-instruct.Q4_K_M.gguf \
  --local-dir ./models

Usage

Ollama runs as a separate service (like Foundry).
After starting, use it as a Custom provider:

http://localhost:11434/v1

Install:

winget install Ollama.Ollama

Start (runs automatically after install):

ollama serve
