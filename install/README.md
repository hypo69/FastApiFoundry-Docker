# 📦 Скрипты установки (Install)

## Описание
Набор вспомогательных инструментов для автоматизации первичной настройки и развертывания проекта.

## Состав пакета

*   **`install-models.ps1`**: Загрузка базовых весов моделей для Foundry и RAG-системы.
*   **`install-shortcuts.ps1`**: Создание ярлыков на рабочем столе для быстрого запуска (стандартный и бесшумный режимы).

## Использование
Данные скрипты вызываются автоматически основным установщиком `install.ps1`, но могут быть запущены отдельно для обновления компонентов.
