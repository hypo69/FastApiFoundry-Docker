# 🧪 Тестирование проекта Ai Assistant

Данная директория содержит все компоненты для обеспечения качества кода, разделенные по уровням ответственности.

## 📂 Структура директории

- **unit/**: Модульные тесты для изолированной проверки функций, классов и отдельных скриптов.
- **integration/**: Проверка взаимодействия между компонентами системы (FastAPI, Foundry, RAG).
- **agents/**: Тестирование логики поведения и инструментов AI-агентов.
- **e2e/**: Сценарии сквозного тестирования пользовательских интерфейсов.

## 🚀 Запуск тестов

Для запуска PowerShell тестов (Pester):
```powershell
Invoke-Pester -Path .\tests\unit\
```

Для запуска Python тестов (pytest):
```powershell
pytest tests/
```
