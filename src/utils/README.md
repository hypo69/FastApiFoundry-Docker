# 🛠️ Вспомогательные утилиты (Utils)

## Описание
Собрание низкоуровневых инструментов для обеспечения единообразной работы с системными ресурсами, форматами данных и консольным выводом.

## Основные компоненты

*   **`json.py`** (`j_loads`, `j_dumps`) — Безопасная работа с JSON-файлами (автоматическое создание папок, логирование ошибок).
*   **`printer.py`** (`pprint`) — Форматированный вывод данных в консоль (замена стандартному `print()`).
*   **`file.py`** — Операции чтения и записи текстовых файлов, управление путями.

## Особенности использования

Согласно `CODE_RULES.md`, прямое использование стандартных библиотек `json` или функции `print()` в проекте **запрещено**. Все операции должны проходить через данные утилиты для обеспечения предсказуемости поведения системы.

---
**Важно**: Любое изменение в `j_loads` или `j_dumps` должно быть протестировано на совместимость со всеми конфигурационными файлами проекта.
