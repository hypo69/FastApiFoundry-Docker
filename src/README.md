# 📦 Директория исходного кода (src)

## Описание
Центральное хранилище исходного кода проекта **AiStros**. Здесь сосредоточена основная бизнес-логика, API-интерфейсы, механизмы интеграции с LLM и вспомогательные инструменты.

## Структура и назначение

*   **`api/`** — Реализация конечных точек (endpoints) FastAPI и логика обработки запросов.
*   **`utils/`** — Набор вспомогательных библиотек для работы с данными, файлами и JSON.
*   **`logger/`** — Конфигурация и реализация системы логирования.
*   **`servers/`** — Скрипты и логика реализации MCP (Model Context Protocol) серверов.
*   **`config/`** — Глобальные настройки проекта и управление состоянием (gs.py).

## Принципы разработки

1.  **Импорты**: Все внутренние модули импортируются относительно корня проекта через `from src...`.
2.  **Устойчивость**: Обязательное использование блоков `try... catch` для обеспечения отказоустойчивости.
3.  **Логирование**: Регистрация всех значимых событий через централизованный логгер.
