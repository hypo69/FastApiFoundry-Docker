# 🌐 API Endpoints

## Описание
Организация сетевого интерфейса взаимодействия с платформой. Директория содержит маршруты FastAPI, схемы валидации данных и логику интеграции с внешними AI-провайдерами (Foundry, llama.cpp, OpenAI).

## Ключевые функции

*   **Маршрутизация**: Определение путей доступа к функционалу AI-ассистента.
*   **Валидация**: Проверка входящих данных на соответствие схемам Pydantic.
*   **Управление потоками**: Обработка асинхронных запросов к моделям инференса.

## Правила
Добавление новых маршрутов требует обязательного обновления документации OpenAPI (Swagger) и включения детальных комментариев на английском языке внутри кода.
