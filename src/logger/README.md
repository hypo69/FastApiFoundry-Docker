# 📝 Система логирования (Logger)

## Описание
Централизованный механизм регистрации событий, ошибок и отладочной информации проекта AiStros. Обеспечивает прозрачность работы всех модулей системы.

## Функционал

*   **Регистрация ошибок**: Запись исключений с полным stack trace.
*   **Управление уровнями**: Поддержка стандартов DEBUG, INFO, WARNING, ERROR.
*   **Ротация файлов**: Автоматическое сохранение логов в директорию `logs/`.

## Использование
Согласно `CODE_RULES.md`, в проекте запрещено использование стандартной функции `print()`. Все уведомления должны проходить через `src.logger.logger`.
