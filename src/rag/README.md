# 🧱 WordPress MCP Server

## Описание
Специализированный сервер для управления сайтами на базе WordPress (включая Multisite) через инструменты MCP.

## Ключевые операции

*   **Управление контентом**: Создание, редактирование и удаление записей, страниц и медиафайлов.
*   **Системное администрирование**: Взаимодействие с `WP-CLI` для выполнения сложных команд обслуживания.
*   **Интеграция с БД**: Выполнение безопасных запросов к таблицам MariaDB с учетом языковых префиксов.
*   **Управление плагинами**: Активация и деактивация расширений по требованию AI.

---
**Зависимости**: WP-CLI, PHP 8.2+
**Безопасность**: Nonce-проверки и Capability-валидация
