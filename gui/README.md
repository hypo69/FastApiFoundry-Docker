# 🖥️ Ai Assistant GUI

Данная директория содержит компоненты графического интерфейса пользователя на базе `pywebview`.

## Структура
- `Start-User.ps1`: Основной скрипт запуска для конечного пользователя.
- `app.py`: Точка входа приложения (Python-оболочка).
- `api.py`: Мост между JavaScript и системными вызовами Python.
- `frontend/`: Веб-ресурсы интерфейса (HTML, JS, CSS).

## Особенности реализации
- **Streaming (SSE)**: Поддержка потоковой отрисовки ответов от LLM.
- **Python Bridge**: Возможность выполнения команд PowerShell напрямую из UI.
- **Zero Local Backend**: UI общается напрямую с внешним FastAPI сервером, не запуская локальный сервер разработки.
