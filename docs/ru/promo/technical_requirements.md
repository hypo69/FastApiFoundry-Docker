# Технические требования к серверу для стабильной работы локальных моделей

## Введение

Эффективная и безопасная работа локальных AI-моделей в нашем интеллектуальном помощнике напрямую зависит от адекватных аппаратных ресурсов. Этот раздел описывает минимальные и рекомендуемые технические требования к серверу или рабочей станции для обеспечения стабильности, производительности и безопасности системы.

## 1. Процессор (CPU)

- **Минимальные требования:** Современный многоядерный процессор (например, Intel Core i5 8-го поколения или AMD Ryzen 5 2000-й серии) с поддержкой инструкций AVX2. Это необходимо для эффективной работы `llama.cpp` и CPU-версий моделей Foundry.
- **Рекомендуемые требования:** Intel Core i7/i9 10-го поколения и выше, или AMD Ryzen 7/9 3000-й серии и выше. Большее количество ядер и высокая тактовая частота значительно ускоряют инференс, особенно для больших моделей без GPU.

## 2. Оперативная память (RAM)

- **Критический параметр:** Объем оперативной памяти является одним из наиболее важных факторов для локальных LLM, поскольку модели загружаются в RAM (или VRAM).
- **Минимальные требования:**
    - 8 ГБ: Для запуска очень маленьких моделей (до 2-3 миллиардов параметров) или для базовой работы системы без активного использования RAG.
    - 16 ГБ: Для комфортной работы с моделями среднего размера (до 7-10 миллиардов параметров) и активным использованием RAG-системы.
- **Рекомендуемые требования:**
    - 32 ГБ и более: Для запуска крупных моделей (13B+ параметров), одновременной работы нескольких моделей, интенсивного использования RAG с большими индексами и для обеспечения запаса на будущее.

## 3. Видеокарта (GPU) и видеопамять (VRAM)

- **Опционально, но крайне желательно:** Наличие дискретной видеокарты с достаточным объемом VRAM значительно ускоряет инференс.
- **Минимальные требования:**
    - 6-8 ГБ VRAM: Для частичной оффлоадинга небольших моделей (до 7B параметров) на GPU (например, с `ngl` в `llama.cpp`).
    - NVIDIA (CUDA): Рекомендуется для лучшей совместимости и производительности.
- **Рекомендуемые требования:**
    - 12 ГБ VRAM и более: Для эффективной работы с моделями среднего и крупного размера, а также для ускорения обработки эмбеддингов в RAG-системе.
    - NVIDIA RTX 30-й или 40-й серии, или аналогичные AMD Radeon.

## 4. Дисковое пространство

- **Минимальные требования:**
    - 20 ГБ свободного места: Для установки системы, зависимостей и нескольких небольших моделей.
    - 50 ГБ свободного места: При активном использовании RAG-системы (индексы могут занимать значительное место) и нескольких моделей среднего размера.
- **Рекомендуемые требования:**
    - 100 ГБ и более: Для хранения большого количества моделей, объемных RAG-индексов и логов. Рекомендуется SSD для быстрой загрузки моделей и доступа к данным.

## 5. Операционная система

- **Основная платформа:** Windows 10/11 (64-бит) с PowerShell 7+.
- **Поддержка Docker:** Для развертывания в контейнеризированной среде.
- **Linux:** Возможность запуска через Docker или нативную установку Python.

## 6. Программное обеспечение

- **Python:** Версия 3.11 или выше.
- **Git:** Для клонирования репозитория и управления версиями.
- **Microsoft Foundry Local (опционально):** Для работы с ONNX-моделями.
- **Ollama (опционально):** Для упрощенного управления GGUF-моделями.

## Заключение

Правильный выбор аппаратного обеспечения является залогом стабильной и производительной работы нашего интеллектуального помощника. Инвестиции в достаточный объем RAM и, при возможности, в GPU, окупятся значительным повышением скорости обработки и расширением возможностей использования AI-моделей.
