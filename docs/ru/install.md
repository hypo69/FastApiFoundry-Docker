## Системные требования
- **ОС**: Windows 10/11 с Windows PowerShell 5.1+ или PowerShell 7+.
- **Python**: 3.11.x.
- **Дисковое пространство**: Минимум **5 ГБ** свободного места на целевом диске для размещения весов LLM моделей.

## Предварительные проверки (Pre-flight Checks)
Перед запуском `install.ps1`, убедитесь в следующем:

1.  **Права доступа**: У текущего пользователя должны быть права на создание папок в домашней директории (`~/.models`, `~/.rag`) и в корне проекта.
2.  **Свободное место**: Интеграционные тесты заблокируют запуск, если на диске менее 5 ГБ свободного пространства. Это критично для предотвращения повреждения файлов моделей при скачивании.
3.  **Интернет**: Для первого запуска требуется доступ к GitHub (загрузка `llama.cpp`) и HuggingFace.

## Процесс установки

1.  Запустите `install.bat` или откройте PowerShell от имени пользователя.
2.  Выполните основной скрипт установки:
    ```powershell
    .\install.ps1
    ```
3.  При появлении запроса выберите источник для `llama.cpp`:
    - `[G]`: Скачать последнюю версию с GitHub (рекомендуется).
    - `[L]`: Использовать локальный архив из папки `bin/`.

## Устранение неполадок
Если установка завершилась с ошибкой доступа, проверьте настройки антивируса или запустите терминал с повышенными привилегиями для выполнения `Set-ExecutionPolicy Bypass`.
