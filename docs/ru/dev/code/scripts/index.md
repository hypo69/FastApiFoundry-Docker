Директория содержит вспомогательные сценарии для обслуживания проекта, управления зависимостями и автоматизации рутинных задач.

## 📄 Состав скриптов

- **Download-LlamaCpp.ps1**: Автоматизация получения последних бинарных файлов `llama.cpp` напрямую из репозитория разработчика (GitHub).
- **Update-Project.ps1**: Проверка и выполнение обновлений текущего репозитория.
- **Get-TelegramChatId.ps1**: Помощник для получения `chat_id` и автоматической настройки уведомлений в `.env`.
- **Invoke-Qa.ps1**: Полный цикл проверок (линтеры, типы, тесты, покрытие, аудит зависимостей и безопасности).
- **Fix-Vulnerabilities.ps1**: Автоматическое исправление (patching) уязвимых зависимостей.
- **Setup-GitHooks.ps1**: Автоматизированная установка git-хуков (например, pre-commit для поиска секретов).
