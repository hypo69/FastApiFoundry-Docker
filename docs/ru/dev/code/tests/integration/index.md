Директория содержит тесты для проверки взаимодействия между компонентами системы и внешними ресурсами (файловая система, процессы, сеть).

## 📄 Состав тестов

- **LlamaExtraction.Tests.ps1**: Проверка жизненного цикла бинарных файлов.
- **test_fastapi_init.py**: Валидация загрузки настроек из `config.json` и их корректности для сервера.
- **test_api.py**: Систематическая проверка API эндпоинтов (Health, Models, Docs).
  - **Генерация**: Тест `/api/v1/generate` с моком LLM через `respx`.
  - **Ошибки**: Проверка кодов 401, 404, 422.
  - **Таймауты**: Обработка задержек бэкенда (504 Gateway Timeout).
- **locustfile.py**: Сценарий нагрузочного тестирования для проверки стабильности под давлением.
  - Проверка существования физических путей, указанных в секции `directories`.
  - Валидация прав доступа (чтение и запись) для рабочих директорий.
  - Проверка свободного места на диске для директории моделей (минимум 5 ГБ).
  - Инициализация FastAPI приложения.
  - Обнаружение zip-архивов в директории `bin/`.
  - Реальная распаковка архива средствами PowerShell.
  - Обновление версии в `config.json` после успешного извлечения.
