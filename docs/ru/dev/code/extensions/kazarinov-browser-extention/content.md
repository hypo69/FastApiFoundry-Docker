### `handleRepeatGeneration`

Глобальная функция для отображения ошибки на странице Preview с кнопкой повтора.
 * 
 *

| Параметр | Тип | Описание |
|---|---|---|
| `message` | `string` | Сообщение об ошибке. |
| `showRepeat` | `boolean` | Показывать ли кнопку "Повторить запрос". |
| `repeatData` | `object|null` | Данные для повтора {tabId, lang}. |
| `tabId` | `number` | ID вкладки, откуда был инициирован запрос (для UI). |
| `lang` | `string` | Код языка. |
