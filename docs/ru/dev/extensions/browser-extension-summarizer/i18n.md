# Многоязычность (i18n)

Механизм переводов UI в расширении browser-extension-summarizer.

## Файлы локализации

Строки UI хранятся в `locales/`:

| Файл | Язык |
|---|---|
| `locales/en.json` | English |
| `locales/ru.json` | Русский |
| `locales/he.json` | עברית |

## Модуль `js/i18n.js`

Загружает строки по текущей локали браузера и подставляет их в элементы с атрибутом `data-i18n`.

```javascript
// Применить переводы ко всем элементам с data-i18n
applyI18n();

// Получить строку по ключу
const label = i18n('btn.summarize');
```

## Добавление нового языка

1. Создать `locales/<code>.json` по образцу `locales/en.json`
2. Добавить код языка в `js/i18n.js` в массив поддерживаемых локалей
3. Добавить промпты в `prompts/<code>.js` (опционально)
