# Установка и первый запуск

## Загрузка в Chrome

1. Открыть `chrome://extensions/`
2. Включить **Режим разработчика** (переключатель в правом верхнем углу)
3. Нажать **Загрузить распакованное расширение**
4. Выбрать папку:
   ```
   FastApiFoundry-Docker/extensions/browser-extension-summarizer/
   ```
5. Расширение появится в списке с иконкой 🤖

## Первичная настройка

После загрузки расширение не имеет настроенных провайдеров. Нужно добавить хотя бы один API-ключ.

### Вариант A — внешний провайдер (Gemini, OpenAI и др.)

1. Нажать иконку расширения в панели Chrome
2. Нажать **⚙️ Providers & Models**
3. Найти нужного провайдера (например **Google Gemini**)
4. Ввести API-ключ в поле → нажать **+ Add key**
5. Нажать **Load models** — список моделей загрузится через API
6. Кликнуть на нужную модель → нажать **Set Active**

### Вариант B — локальный FastAPI Foundry сервер

1. Убедиться что сервер запущен: `http://localhost:9696/api/v1/health`
2. Открыть **⚙️ Providers & Models**
3. Найти провайдер **Custom (OpenAI-compatible)**
4. В поле **Base URL** ввести: `http://localhost:9696/v1`
5. Поле API-ключ оставить пустым (или ввести если задан `API_KEY` в `.env`)
6. Нажать **Load models** — появятся модели с локального сервера
7. Выбрать модель → **Set Active**

## Проверка работы

### Суммаризация страницы

1. Открыть любую веб-страницу
2. Правый клик → **Summarise** → **Only This Page**
3. Откроется новая вкладка с прогрессом и результатом

### Чат

1. Нажать иконку расширения → **💬 Chat**
2. Откроется страница чата в новой вкладке
3. Ввести сообщение → Enter

### Отладка сервера

1. Нажать иконку расширения → **🔧 Server Debug**
2. Убедиться что URL сервера правильный (`http://localhost:9696`)
3. Нажать **Check** — статус должен показать **✓ Connected**

## Обновление расширения

После изменения файлов расширения:

1. Открыть `chrome://extensions/`
2. Нажать кнопку обновления (🔄) на карточке расширения
3. Или нажать **Ctrl+R** на странице `chrome://extensions/`

!!! note "Настройки сохраняются"
    `chrome.storage.sync` хранит все ключи и настройки. Они не сбрасываются при обновлении расширения.

## Разрешения

| Разрешение | Назначение |
|---|---|
| `contextMenus` | Пункт меню «Summarise» при правом клике |
| `activeTab` | Доступ к текущей вкладке для извлечения текста |
| `tabs` | Список всех вкладок для режима All Open Tabs |
| `scripting` | Выполнение скрипта для извлечения текста страницы |
| `storage` | Хранение настроек, ключей и данных задач |
| `notifications` | Уведомления об ошибках |
| `host_permissions: <all_urls>` | Извлечение текста с любых сайтов |
| `host_permissions: *.googleapis.com` и др. | Запросы к API провайдеров |

## Требования

- Chrome 88+ (поддержка Manifest V3 и ES Modules в service worker)
- Для FastAPI Foundry: запущенный сервер на `localhost:9696`
- Для внешних провайдеров: действующий API-ключ
