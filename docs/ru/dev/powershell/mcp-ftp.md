# ftp_mcp.py

**Файл:** `mcp/src/servers/ftp_mcp.py`  
**Протокол:** MCP STDIO  
**Версия:** 0.6.1

## Назначение

MCP сервер для работы с FTP серверами. Параметры подключения читаются из переменных окружения (`.env`).

## Инструменты

| Инструмент | Описание |
|---|---|
| `ftp_list` | Список файлов в директории |
| `ftp_upload` | Загрузить локальный файл на FTP |
| `ftp_download` | Скачать файл с FTP |
| `ftp_delete` | Удалить файл на FTP |
| `ftp_rename` | Переименовать / переместить файл на FTP |

## Переменные окружения

| Переменная | Описание |
|---|---|
| `FTP_HOST` | Хост FTP сервера |
| `FTP_USER` | Имя пользователя |
| `FTP_PASSWORD` | Пароль |
| `FTP_PORT` | Порт (по умолчанию: 21) |

## Запуск

```powershell
venv\Scripts\python.exe mcp/src/servers/ftp_mcp.py
```
