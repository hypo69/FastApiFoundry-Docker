# Чат с моделью

Основная вкладка для общения с AI — **Chat**.

---

## Интерфейс чата

- Поле ввода внизу — введите вопрос, отправьте по `Enter`
- Новая строка без отправки — `Shift+Enter`
- История диалога сохраняется в рамках сессии
- Кнопка **Clear** — очистить историю и начать новый диалог

---

## Параметры ответа

| Параметр | Что делает | Рекомендация |
|---|---|---|
| **Model** | Какая модель отвечает | Выберите один раз, менять не нужно |
| **Temperature** | Случайность ответа (0–2) | 0.3–0.7 для точных ответов, 0.8–1.2 для творческих |
| **Max Tokens** | Максимальная длина ответа | 512 для коротких, 2048+ для развёрнутых |
| **Use RAG Context** | Искать в базе знаний перед ответом | Включите, если у вас есть индекс документов |
| **Locale** | Язык ответа | Оставьте пустым — модель ответит на языке вопроса |

---

## Потоковый вывод

Ответ появляется постепенно, слово за словом — это нормально. Модель генерирует текст в реальном времени.

---

## RAG в чате

Если включён **Use RAG Context**, система перед генерацией ответа:

1. Ищет релевантные фрагменты в вашем индексе документов
2. Добавляет найденное в контекст запроса
3. Модель отвечает, опираясь на ваши документы

Подробнее: [База знаний (RAG)](rag_user.md)
