# Начало работы с FastAPI Foundry

FastAPI Foundry — это мощная платформа для интеграции моделей искусственного интеллекта (ИИ) с веб-приложениями на базе FastAPI. Она позволяет легко запускать локальные модели ИИ, управлять ими и предоставлять к ним доступ через удобный API и веб-интерфейс.

## 🚀 Основные возможности

*   **Интеграция с Foundry Local / llama.cpp / Ollama**: Поддержка различных бэкендов для запуска локальных ИИ-моделей.
*   **Гибкая конфигурация**: Управление настройками через `config.json` и переменные окружения.
*   **Веб-интерфейс**: Интуитивно понятный интерфейс для взаимодействия с моделями и RAG-системой.
*   **Система RAG (Retrieval-Augmented Generation)**: Поиск и извлечение контекста из ваших документов для улучшения ответов ИИ.
*   **Агенты ИИ**: Расширяемые агенты, способные использовать инструменты для выполнения сложных задач.

## 💻 Быстрый старт (Quick Launch)

Самый простой способ запустить FastAPI Foundry — использовать скрипт `launcher.ps1` и выбрать "Quick Launch". Этот режим автоматически устанавливает все необходимые зависимости и запускает сервер.

1.  **Запустите `launcher.ps1`**:
    ```powershell
    .\launcher.ps1
    ```
2.  **Выберите "1. 🎯 Quick Launch (Recommended)"**:
    Следуйте инструкциям на экране. Скрипт проверит наличие Python, создаст виртуальное окружение, установит зависимости, настроит `.env` и предложит установить Microsoft Foundry Local.

3.  **Откройте веб-интерфейс**:
    После успешного запуска сервера, откройте в браузере:
    [http://localhost:9696](http://localhost:9696) (или другой порт, если настроена автоматическая смена)
