# База знаний (RAG)

RAG (Retrieval-Augmented Generation) — возможность подключить свои документы, чтобы модель отвечала на основе ваших данных, а не только встроенных знаний.

Простая аналогия: **экзамен с открытой книгой**. Модель получает вопрос, находит нужные страницы в вашей «библиотеке» и формулирует ответ на их основе. Ваши данные никуда не отправляются.

---

## Что можно использовать как источник знаний

- **Офисные документы**: Word, Excel, PowerPoint (включая старые форматы)
- **PDF**: обычные и отсканированные (OCR)
- **Архивы**: `.zip`, `.7z`, `.rar`, `.tar` — система сама распакует и проиндексирует
- **Текстовые файлы**: `.txt`, `.md`, письма `.eml`, `.msg`
- **Веб-страницы**: дайте ссылку — система извлечёт текст
- **Код**: 40+ языков программирования

---

## Загрузить документы

Перейдите на вкладку **RAG → Index Documents**.

### Загрузить файлы или архив

1. Выберите вкладку **ZIP / Files**
2. Нажмите на поле выбора файлов и выберите один или несколько файлов
3. Нажмите **Upload & Index**
4. Прогресс отображается для каждого файла: извлечение → нарезка → эмбеддинги → индекс

### Загрузить папку целиком

1. Выберите вкладку **Directory**
2. Нажмите на поле выбора — откроется системный диалог выбора папки
3. Выберите папку — браузер передаст все файлы внутри (включая подпапки)
4. Нажмите **Index Directory**

!!! tip "Время индексации"
    Небольшая папка (до 100 файлов) — несколько секунд.
    Большой архив (тысячи файлов) — может занять минуты.
    Прогресс отображается в реальном времени.

## Использовать базу знаний в чате

1. Перейдите на вкладку **Chat**
2. Включите чекбокс **Use RAG Context**
3. Задайте вопрос — модель будет искать ответ в ваших документах

---

## Управление документами

Вкладка **RAG → Document Manager** показывает все проиндексированные документы.

- **Редактировать** — изменить текст, переиндексируются только изменённые части
- **Переиндексировать** — принудительно пересчитать эмбеддинги
- **Удалить** — убрать документ из базы знаний
- **Compact** — очистить индекс от удалённых документов (появляется автоматически)

---

## Несколько баз знаний

Если настроено несколько RAG-профилей, переключайтесь между ними на вкладке **RAG → RAG Bases**:

- Кнопка ▶ — активировать профиль
- Кнопка ⏸ — отключить RAG
- Кнопка 🗑 — удалить профиль
