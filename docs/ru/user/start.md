# Начало работы

Система уже запущена. Откройте браузер и перейдите по адресу:

**http://localhost:9696**

Если адрес недоступен — обратитесь к администратору системы.

---

## Что вы увидите

Веб-интерфейс с вкладками в верхней части страницы. Индикатор в шапке показывает статус:

| Индикатор | Значение |
|---|---|
| 🟢 Connected | Система работает, модель готова |
| 🟡 API Only | API работает, AI-бэкенд недоступен |
| 🔴 Error | Сервер недоступен |

---

## Первый запрос

1. Перейдите на вкладку **Chat**
2. Убедитесь, что в поле **Model** выбрана модель (не пусто)
3. Введите вопрос в поле внизу
4. Нажмите **Enter** или кнопку отправки

Ответ появится в области диалога.
