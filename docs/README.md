# 📖 Документация проекта Ai Assistant

Данная директория содержит подробные руководства по установке, настройке и обеспечению качества проекта.

## 📂 Структура

- **ru/**: Русскоязычная версия документации.
  - `install.md`: Инструкции по развертыванию и системные требования.
  - `qa.md`: Описание системы тестирования и регламент проверок.

Актуальные отчеты о покрытии кода генерируются автоматически в `tests/reports/`.
